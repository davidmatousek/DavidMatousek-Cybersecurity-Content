## Frameworks

### Security Manifesto

A 10-principle security framework for AI-assisted development. Designed to integrate with Claude Code (or any AI coding assistant) to enforce secure-by-default practices across every code generation and file operation.

**Covers**: memory safety, input validation, secure defaults, security logging, least privilege, encryption, fail-secure patterns, dependency security, secret management, and security testing integration.

- [Security Manifesto](./SecurityManifesto/SecurityManifesto.md) — Full framework
- [Integration Guide](./SecurityManifesto/Readme.md) — Setup instructions for Claude Code projects

### The Seven Strategic Cybersecurity Posture Domains

A strategic framework for organizational security posture assessment and improvement. Provides a structured approach to evaluating and strengthening security across seven key domains.

- [Framework (PDF)](./The%20Seven%20Strategic%20Cybersecurity%20Posture%20Domains/The%20Seven%20Strategic%20Cybersecurity%20Posture%20Domains%20v1.0.pdf)
